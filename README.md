# Cats & Dogs

This [Telegram](https://telegram.org/) bot can show you some cats and/or dogs.

## Setup

- Install Python 3 on your machine
- pip install python-telegram-bot --upgrade
- Get a token from [@Botfather](https://telegram.me/botfather)
